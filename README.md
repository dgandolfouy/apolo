## Cómo Iniciar
1. Instalar dependencias:
   ```bash
   npm install
   ```
2. Iniciar servidor de desarrollo:
   ```bash
   npm run dev
   ```

## Estructura
- `src/`: Código fuente React + Vite.
- `public/`: Assets estáticos.
- `vite.config.ts`: Configuración del bundler.
